# منصة إنشاء البراند العطري الاحترافية

## 1. وصف المشروع
منصة متكاملة لإدارة مشاريع إنشاء البراندات العطرية، تعمل كموقع تعريفي احترافي مع CTA للتسجيل. بعد التسجيل ينتقل العميل لصفحة الخدمات المستقلة المرتبطة بلوحة التحكم، حيث يمكنه اختيار الخدمة المناسبة ومتابعة مشاريعه. الموقع باللغة العربية مع دعم RTL كامل.

## 2. هيكل الصفحات

### الصفحة الرئيسية (`/`) — موقع تعريفي احترافي (6+ أقسام)
1. **Navbar** — شريط تنقل ذكي مع قائمة منسدلة للخدمات: الرئيسية | الخدمات (قائمة منسدلة بـ 6 خدمات) | عنا | نماذج من الأعمال | الأسئلة والاستفسارات | تواصل معنا | التسجيل
2. **ServiceHero** — هيرو احترافي مع وسوم المسارات وإحصائيات المنصة و CTA للتسجيل
3. **AboutSection** — قسم "عن المنصة" مع إحصائيات + قصة المنصة + القيم الأربعة
4. **ServicesShowcase** — المسارات الستة بتصميم فاخر (خلفية داكنة)
5. **PortfolioSection** — نماذج من الأعمال مع فلترة حسب الفئة
6. **FAQSection** — الأسئلة الشائعة (8 أسئلة) مع أكورديون تفاعلي
7. **ContactSection** — تواصل معنا: معلومات اتصال + فورم إرسال
8. **ArtisticCTA** — دعوة للإجراء النهائية
9. **FooterSection** — تذييل مع روابط ونشرة بريدية

### الصفحات العامة (لا تحتاج تسجيل)
- `/auth` - تسجيل الدخول / التسجيل → بعد التسجيل ينتقل لـ `/services`
- `/services` - صفحة الخدمات المستقلة (مرتبطة بلوحة التحكم)
- `/services/:id` - تفاصيل المسار (A-F)

### مساحة العميل (تحتاج تسجيل دخول)
- `/dashboard` - لوحة التحكم (زر "مشروع جديد" ← `/services`)
- `/dashboard/projects` - المشاريع
- `/dashboard/projects/manage` - إدارة المشاريع (كانبان)
- `/dashboard/messages` - الرسائل
- `/dashboard/approvals` - الاعتمادات
- `/dashboard/invoices` - الفواتير
- `/dashboard/settings` - الإعدادات
- `/workspace` - مساحة عمل العميل

### مراحل الخدمة (تحتاج تسجيل دخول)
- `/brand-creation` - بناء البراند
- `/visual-identity` - الهوية البصرية
- `/packaging-customizer` - مخصص العبوة
- `/studio-3d` - معاينة 3D
- `/manufacturing` - التصنيع
- `/pricing` - حاسبة التسعير
- `/production` - تطوير العطر
- `/after-sales` - ما بعد البيع
- `/workflow` - سير العمل

## 3. تدفق المستخدم
1. **زائر** ← يتصفح الصفحة الرئيسية (أقسام تعريفية) ← يضغط CTA "التسجيل"
2. **يسجل** ← ينتقل مباشرة لصفحة `/services`
3. **يختار خدمة** ← ينتقل لصفحة الخدمة المحددة (تحتاج تسجيل)
4. **لوحة التحكم** ← زر "مشروع جديد" ← `/services` لاختيار خدمة جديدة
5. **يتابع** ← من لوحة التحكم يتابع المشاريع والرسائل والاعتمادات

## 4. المسارات الخدمية الستة
1. **المسار A** — براند كامل (Full Brand) → `/brand-creation`
2. **المسار B** — تطوير عطر (Fragrance) → `/production`
3. **المسار C** — تغليف وعبوة (Packaging) → `/packaging-customizer`
4. **المسار D** — هوية بصرية (Identity) → `/visual-identity`
5. **المسار E** — تصنيع (Manufacturing) → `/manufacturing`
6. **المسار F** — طباعة وتنفيذ (Print) → `/services/F`
